# 🤖 Центр Красок №1 — Telegram Бот

Telegram-бот ассистент для компании «Центр Красок №1»

---

## 📋 О проекте

Этот бот отвечает на вопросы клиентов о компании «Центр Красок №1»: контакты, адреса, услуги, бренды, доставка, вакансии.

**Особенности:**
- ✅ Простой чат без команд
- ✅ Интерактивные кнопки меню
- ✅ Поиск по базе знаний
- ✅ Красивое форматирование ответов
